# Juego del Número Secreto

### 📝 Descripción
Desafío de lógica en Python donde el usuario adivina un número aleatorio. El sistema ajusta el objetivo al múltiplo de 3 más cercano y ofrece pistas direccionales. Implementa manejo de errores con try/except.

### 🚀 Tecnologías utilizadas
* **Librerías:** `random`.
* **Conceptos:** Algoritmos de búsqueda, lógica de divisibilidad y experiencia de usuario interactiva por consola.
